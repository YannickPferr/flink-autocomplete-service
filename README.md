# flink-autocomplete-service

## Installation
```
# clone the repo
git clone https://github.com/YannickPferr/flink-autocomplete-service.git
cd flink-autocomplete-service
# activate virtual env
python -m venv venv
source venv/bin/activate
# install packages
pip install -r requirements.txt
```

## Usage
```
# start the server
python server.py
```
